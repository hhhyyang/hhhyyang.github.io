修改ubuntu用戶名

修改前的准备：

vim编辑器：

  • 执行vim filename进入编辑器（默认命令模式）
  • 点击a或i进入编辑模式，输入内容
  • 输入完成后使用Esc退出编辑模式（进入命令模式）
  • 输入：__wq+回车__（w保存并q退出）

1-更改主机名字

修改hostname文件：

修改LittleGreen为想要更改的主机名

此文件中的内容用于显示主机的名字，修改后如果重启，终端中@后面的主机名将会改变

修改hosts文件：

将LittleGreen修改为想要更改的名称

更改完成后，建议重启计算机，否则命令执行会较慢

2-更改用户名

修改sudoer文件：

在红框内另外__添加__想要更改的用户名,格式与前两个相同

修改shadow文件：

此文件中存储与登录相关的内容，格式如下：

username: passwd: lastchg: min: max: warn: inactive: expire: flag

登录名：加密口令：最后一次修改时间：最小时间间隔：最大时间间隔：警告时间：不活动时间：失效时间：标志

将红框内自己原来的用户名修改为欲修改的名字

修改开始目录：

修改passwd文件：

passwd文件内容格式如下：

用户名：密码：uid：gid：用户描述：主目录：登录shell

将用户名、用户描述、主目录修改为新的名字

修改原用户下文件所属的组：

将红框内的用户名修改为欲修改的名字

再次进入sudoer文件，删除原来的用户名

可能遇见的问题：

1-出现如下界面：

有文件未关闭

解决方法：

按‘e’+回车

2-E45: ‘readonly’ option is set (add ! to override)

当前用户没有权限对文件进行修改

解决方法：

输入__:w !sudo tee %__

  • :w—-write
  • !sudo—-call shell sudo command
  • tee—-the output of write(:w)command is redirected using tee
  • %—-current file name
